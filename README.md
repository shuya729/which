## 概要

**BiPick** は、誰もが楽しめる 2 択の質問共有サービスです。恋愛や趣味、日常生活に関する話題など、さまざまなジャンルで究極の選択を気軽に投稿・閲覧することができます。

例えば、

- **「旅行なら海派？山派？」**
- **「人生で一度だけ戻れるとしたらどの瞬間？」**

といった幅広いテーマの 2 択質問が集まり、気になる質問に回答することで、他のユーザーがどちらを選んだのかリアルタイムで結果を確認できます。話題のきっかけにも最適です！
